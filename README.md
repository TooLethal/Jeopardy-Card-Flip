A fun **HTML + CSS project** that recreates a **Jeopardy-style trivia card**. The card displays a question on the front and flips to reveal the answer on hover, using smooth 3D transform animations.

## ‚ú® Features
- **Interactive Flip Animation**  
  - Front side shows the trivia question.  
  - Back side reveals the answer when hovered.  
  - Smooth `rotateY` transitions with `backface-visibility` for realistic flipping.  

- **Jeopardy Theme Styling**  
  - Background color: Deep Jeopardy Blue (`#060CE9`).  
  - Text: White, uppercase, with letter spacing for dramatic effect.  
  - Font: *Times New Roman* for a classic trivia look.  

- **Customizable Variables**  
  - Colors defined in `:root` for easy adjustments.  
  - Card dimensions and font styles can be tweaked for different layouts.  

- **Responsive Centering**  
  - Flexbox centers the card both vertically and horizontally on the page.  

- **Minimal Structure**  
  - Simple HTML with `.card`, `.card-front`, and `.card-back` elements.  
  - Compact CSS for styling and hover animations.
